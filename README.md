# [Titan](https://usetitan.app) for Open Source

This GitHub organization hosts the open-source SDKs and integrations that Titan provides.

Titan also helps maintain multiple open-source packages, including [Baileys](https://github.com/WhiskeySockets/baileys) and [whatsmeow](https://github.com/tulir/whatsmeow). Titan is committed supporting the open-source community. We also offer a permanent free open-source plan for maintainers.

## What's next?

We are piloting a program called "DevCenter" which allows developers to do more in the way of instrumenting the WA protocol.
